Que es Linux?

GNU es un sistema operativo de tipo Unix, lo cual significa que se trata de una colección de muchos programas: aplicaciones, bibliotecas, herramientas de desarrollo y hasta juegos. El desarrollo de GNU, iniciado en enero de 1984, se conoce como Proyecto GNU. Muchos de los programas de GNU se publican bajo el auspicio del Proyecto GNU y los llamamos paquetes de GNU.

El nombre GNU es un acrónimo recursivo de "GNU No es Unix". GNU se pronuncia en inglés como una sílaba sin vocal entre la g y la n. En español se puede pronunciar de la misma manera, o bien reemplazando gn por una ñ, como en ñu. También es aceptable en español la pronunciación de cada una de las letras por separado: G-N-U (ge-ene-u)

En un sistema de tipo Unix, el programa que asigna los recursos de la máquina y se comunica con el hardware se denomina kernel (o núcleo). GNU se usa generalmente con un kernel llamado Linux. Esta combinación es el sistema operativo GNU/Linux. Millones de personas usan GNU/Linux, aunque muchos lo llaman erróneamente Linux.

Instalacion

Aunque haya muchas distribuciones de linux, la mayoria de ellas se pueden instalar muy similar a los demas.

Para instalar linux, se necesita las siguientes cosas:

  • Una USB (Recomendado 8 o 16gbs)
  • El archivo ISO de la distribucion que desea instalar
  • Un software de creacion de USBs Liveboot (En este caso, utilizaremos Balenaetcher)
    •

Para este tutorial, utilizaremos Linux Mint como la distribucion que queremos instalar en nuestro sistema.

Paso 1: Obtener el ISO

  1. Ir a la pagina oficial de Linux Mint (linuxmint.com) y entrar a la pagina de Descargar
  2. Escoger el entorno grafico que desea (Para mas detalles, describo la diferencias entre las 3 en la pagina de Caracteristicas de este sitio web)
  3. Escoger uno de los links de descargar disponibles al pie de la pagina. Tambien hay archivos torrent si son necesisarios.
  4. Instalar una herramienta para instalar archivos .iso (Como el de Linux Mint) a una memoria USB (Yo recomendaria BalenaEtcher)

Paso 2: Instalar el ISO en una USB

Para instalar linux en una computadora, necesitamos poner el iso que descargamos en una memoria USB. Hay varias herramientas para hacer esto pero yo recomendaria Balenaetcher

El proceso de instalar Mint en una USB es muy facil con BalenaEtcher.

  1. Se escoger el archivo .iso de mint
  2. Se escoge el dispositivo USB en donde se va a instalar mint
  3. Se comfirma la accion

El proceso puede tomar varios minutos, dependiendo en la velocidad de la Memoria USB y la computadora que esta corriendo la aplicacion.

Paso 3: Correr la USB en la computadora

Se tiene que primero que todo apagar la computadora donde se quiere instalar Linux Mint y conectar la USB a uno de los pueros USB de la computadora.

Despues de tiene que entrar al BIOS de la computadora. El BIOS se puede explicar como una pantalla de configuracion del hardware directamente de la tarjeta madre. Para entrar el BIOS se tiene presionar una tecla (Cual tecla depende en el vendedor de la computadora o tarjeta madre, normalmente puede ser la tecla F12 o de punto (".") ) al momento de prender la computadora.

Su BIOS puede parecer diferente a este, pero todos tienen los mismos apartados.

En su BIOS tiene que buscar el apartado que mencione "Boot" o algo parecido. Normalmente cuando usted encuentra esta opcion, por default, va a decir que bootea la computadora desde el disco duro instalado. Esto se tiene que cambias a Dispositivos Retirables (Como USBs)

Cuando se cambie esta opcion, se tiene que guardar los cambios y salir del BIOS.

Paso 4: Corriendo la distro por primera vez

Este paso puede depender en el distro que se escogio para instalar, pero la mayoria siguen una orden similar para facilitar la instalacion para usuarios.

En el caso de Mint, se tiene que hacer los siguientes pasos:

  1. Escoger la opcion de "Start Mint" cuando aparesca el menu al correr la computadora
  2. Despues de llegar al escritorio del sistema, escoja la opcion de "Instalar Linux Mint"
  3. Escoger el idioma que quiere utilizar en el sistema
  4. Escoger formato de teclado
  5. Escoger si quieres descargar los archivos necesarios para reproducir varios tipos de archivos multimedia (Videos, imegenes, audio, etc)
  6. Escoger cual es el disco duro donde se va a instalar mint
  7. Escoger zona de tiempo
  8. Escribir el nombre de usuario y contraseña que usted desea

Despues de comfirma todo esto, se va comenzar la instalacion completa de mint a su sistema.

Esto puede tomar un tiempo, y depende mucho en la velocidad de la Memoria USB en donde usted instalo Mint y la velocidad de la computadora misma que esta corriendo la USB

Despues de terminar la instalacion, se le va a preguntar al usuario que reinicie la computadora y cuando prenda la computadora otra vez, ya tendra Linux Mint instalado en su sistema.
